# Task Manager App

A modern, responsive, and feature-rich Kanban-style Task Management application built with React. This application allows users to organize their daily routines, project progress, and personal development tasks across multiple boards.

## 🚀 Live Demo
Check out the live application: [Task Manager App](https://task-manager-app-sandy-seven.vercel.app/)

![Task Manager Screenshot](public/screenshot.png)

## ✨ Features

- **Dynamic Boards**: Create and manage multiple boards for different projects or routines.
- **Kanban Workflow**: Organize tasks into customizable columns (e.g., Morning, Afternoon, Evening).
- **Task Management**: Add, view, edit, and delete tasks with ease.
- **Subtasks Tracking**: Track progress of subtasks within each main task.
- **Theme Switching**: Seamlessly toggle between Light and Dark modes.
- **Responsive Design**: Fully optimized for both desktop and mobile viewing.
- **Interactive UI**: Smooth transitions and modals for a premium user experience.

## 🛠️ Technologies Used

- **Frontend**: [React](https://reactjs.org/) + [Vite](https://vitejs.dev/)
- **State Management**: [Redux Toolkit](https://redux-toolkit.js.org/)
- **Styling**: [Tailwind CSS](https://tailwindcss.com/)
- **UI Components**: [Headless UI](https://headlessui.dev/)
- **Icons**: Custom SVG icons and Heroicons
- **Deployment**: [Vercel](https://vercel.com/)

## 📦 Installation

To run this project locally, follow these steps:

1. Clone the repository:
   ```bash
   git clone https://github.com/sanjukumari-tech/taskManager-trello-.git
   ```
2. Navigate to the app directory:
   ```bash
   cd taskManager-app
   ```
3. Install dependencies:
   ```bash
   npm install
   ```
4. Start the development server:
   ```bash
   npm start
   ```

## 🏗️ Build

To create a production build:
```bash
npm run build
```
